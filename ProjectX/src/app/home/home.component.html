Traditionally, the new investors would invest equally in their target companies however the assets can co-move with each other and some of the assets might be negatively correlated with each other. Therefore, we need to find the optimum portfolio. The optimum portfolio is the one that generates the highest return for the lowest risk. It can be overwhelming to generate every single possible portfolio. As a consequence, we can also use use the Monte-Carlo methodology to generate 10'000+ different portfolios in Python. The methodology revolves around the concept that as we increase the number of portfolios, we will get closer to the actual optimum portfolio.

So in Monte Carlo we randomly assign weights to the stocks/assets (here 5) in our portfolio and returns from the portfolio and its volatility are calculated through some formulas that in tern calculates the Sharpe Ratio which help investors understand the return of an investment compared to its risk.
